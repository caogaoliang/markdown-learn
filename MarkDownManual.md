## 3. Markdown 列表

* 无序列表

```Markdown
* 无序列表 1
* 无序列表 2
    * 无序列表 2.1
```

* 无序列表 1
* 无序列表 2
    * 无序列表 2.1
    * 无序列表 2.2

---

* 有序列表

```Markdown
1. 有序列表 1
2. 有序列表 2
3. 有序列表 3
    3.1 有序列表
```

1. 有序列表 1
2. 有序列表 2
3. 有序列表 3
